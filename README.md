Introdução
O presente projeto teve como objetivo o desenvolvimento de um barco controlado remotamente (RC) utilizando a plataforma Arduino Uno, com o intuito de integrar funcionalidades de controlo, monitorização e segurança. Este projeto combina conceitos de eletrónica, programação e robótica educativa, permitindo ao utilizador operar o barco de forma remota e obter informações em tempo real sobre o seu desempenho.
O barco foi concebido com motores DC responsáveis pela propulsão e um servo motor para controlar o leme, permitindo a alteração da direção de forma precisa. A comunicação é realizada através de Bluetooth, possibilitando o envio de comandos para movimentos e ajustes do leme, assim como a leitura de parâmetros como a tensão da bateria e a distância percorrida.
Entre as funcionalidades implementadas destacam-se:
•	Odometer, que estima a distância percorrida pelo barco com base no tempo de funcionamento dos motores;
•	Monitorização da bateria, permitindo avaliar o estado de carga e alertar em situações de tensão baixa;
•	Movimento suave do leme, garantindo maior estabilidade durante a navegação;
•	Acumulação de distância total e por sessão, possibilitando o registo de várias viagens.
Este projeto não só proporciona uma experiência prática de controlo de veículos RC, como também permite compreender conceitos fundamentais de engenharia elétrica, programação e sistemas de controlo, demonstrando a aplicabilidade de tecnologias acessíveis no desenvolvimento de sistemas autónomos e sem fios.
